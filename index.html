What are we doing here?

Our goal is to run discord bots on every computer architecture there is.

Why?

It came from another server where almost every user had their own bot, we also liked to play around with weird machines. It turned out to be a nice benchmark for a given systems usability; running discord bot requires ability to run arbitrary, unsigned code, having access to internet (working network stack) and being able to run python/nodejs or having proper c++ toolchain available. Therefore, if machine is able to run a discord bot, it's also able to run more usable stuff.
